This window has 2 main states:
- History state
- Fav state
in each of these states you will find 6 main points of interest. 6 of these points are buttons and 1 is a data field. The data field has 2 variants referring to the 2 main states of the window. 
# States

## History State

When this state is active the data field will show the copy history of the user. When one of the items in this field is left clicked the item will be copied and will move to the top of the list. When a item is right clicked a menu will appear with :
 - copy
	 When this is pressed the object will be copied.
 - add to Vault 
	 When this is pressed there will be added a file to the main vault with the content of this object in. 
 - delete
	 When this is pressed the object will be deleted out of the list.
## Fav State

When this state is active the data field will show the hidden map favorites in the vault in the data field. When one of the items in this field is left clicked the item will be copied. When a item is right clicked a menu will appear with :
- copy
	 When this is pressed the object will be copied.
- edit 
	 When this is pressed this object will be opened in the [[Editing Window]].
- un Fav
	 When this is pressed the object will be deleted out of the favourite vault.
- open in vault
	 When this is pressed the vault window will open on the location of this object.
# Buttons 

## History button

When this button is pressed the state of this window will change to the History state.
## Fav button

When this button is pressed the state of this window will change to the Fav state.
## Account button

When this button is pressed this window will close and the [[Account Window]] will open up.
## Settings button 

When this button is pressed this window will close and the [[Settings Window]] will open up.
## Vault button

When this button is pressed this window will close and the [[Vault Window]] will open up.
